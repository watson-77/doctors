Плагин для создания каталога врачей с фильтрацией по городам и специализациям.

== Описание ==

Apicoder Doctors - это профессиональный плагин для WordPress, который позволяет создавать и управлять каталогом врачей. Плагин предоставляет удобную систему для отображения информации о врачах с возможностью фильтрации по городам и специализациям.

**Основные возможности:**

*   Создание записей о врачах с фотографиями
*   Организация врачей по городам
*   Классификация по медицинским специализациям
*   Умная система фильтрации
*   Адаптивный дизайн для мобильных устройств
*   Возможность установки демо-данных для тестирования
*   Поддержка стандартных функций WordPress (изображения, поиск и т.д.)

== Установка ==

1. Загруните папку `apicoder-doctors` в каталог `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Перейдите в меню 'Доктора' для управления контентом

**Быстрый старт:**

После активации плагина вы можете установить демо-данные, нажав на ссылку "Установить демо (30 записей)" рядом с кнопкой активации плагина.

== Использование ==

**Создание врачей:**

1. Перейдите в меню "Доктора" → "Добавить нового врача"
2. Заполните информацию о враче:
   * ФИО врача (автоматически заполняется из произвольного поля)
   * Описание и краткое описание
   * Установите изображение врача (миниатюру записи)
3. В правой колонке выберите города и специализации
4. Заполните произвольные поля:
   * Стаж работы (в годах)
   * Стоимость приема (в рублях)
   * Рейтинг (от 1.0 до 10.0)

**Управление таксономиями:**

*   **Города:** Доктора → Города (добавление/редактирование городов)
*   **Специализации:** Доктора → Специализации (добавление/редактирование специализаций)

**Отображение на сайте:**

*   Архив врачей доступен по адресу: `ваш-сайт/doctors/`
*   Отдельная страница врача: `ваш-сайт/doctors/имя-врача/`
*   Архивы по городам: `ваш-сайт/city/город/`
*   Архивы по специализациям: `ваш-сайт/specialization/специализация/`

== Шорткоды ==

Плагин не использует шорткоды. Все данные отображаются через стандартные шаблоны WordPress.

== Шаблоны ==

Плагин использует следующие шаблоны:

1. **archive-doctors.php** - Архивная страница всех врачей с фильтрами
2. **single-doctors.php** - Страница отдельного врача

Вы можете переопределить эти шаблоны в теме вашего сайта, создав файлы в папке:
`wp-content/themes/ваша-тема/`

Пример структуры для переопределения:
- `/your-theme/archive-doctors.php`
- `/your-theme/single-doctors.php`

== Фильтры ==

На странице архива врачей доступны фильтры:
* Фильтр по городу
* Фильтр по специализации
* Кнопка сброса фильтров

Фильтры работают без перезагрузки страницы (AJAX).

== Демо-данные ==

Плагин включает функцию установки демо-данных:

1. Перейдите в "Плагины" → найдите "Apicoder Doctors"
2. Нажмите "Установить демо (30 записей)"
3. Будет создано 30 тестовых записей врачей с:
   * Случайными именами и фамилиями
   * Разными городами и специализациями
   * Реалистичными данными (стаж, цена, рейтинг)

Для удаления демо-данных нажмите "Удалить демо" рядом с плагином.

== Часто задаваемые вопросы ==

= Как добавить изображение врачу? =
После создания записи врача, в правой колонке редактора найдите блок "Изображение записи" и установите изображение.

= Как изменить порядок отображения врачей? =
Используйте стандартный функционал WordPress для сортировки записей или установите плагин для кастомной сортировки.

= Можно ли изменить поля информации о врачах? =
Да, вы можете добавить дополнительные произвольные поля через стандартный функционал WordPress или с помощью плагинов для работы с произвольными полями.

= Как изменить дизайн страниц врачей? =
Создайте файлы `archive-doctors.php` и `single-doctors.php` в вашей теме и отредактируйте их.

= Плагин поддерживает мультиязычность? =
Да, плагин готов для перевода и использует функции интернационализации WordPress.
